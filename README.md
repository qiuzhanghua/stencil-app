# Stencil App Starter

Ready for stencil 3

## Getting Started

To get started, clone this repo to a new directory:

```bash
git clone https://github.com/qiuzhanghua/stencil-app.git my-app
cd my-app
```

Then, install the dependencies:

```bash
pnpm install
```

Finally, run the app in development mode:

```bash
pnpm start
```
